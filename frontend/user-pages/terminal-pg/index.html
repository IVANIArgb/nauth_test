Главная > Терминал

Введите название команды в поле выше и нажмите «Выполнить» (клавиша Enter не запускает команду).

Доступные команды

change-role-admin
Роль admin в БД для текущего пользователя
change-role-super-admin
Роль super_admin в БД
seed-test-data
Тестовые категории, курсы и уроки (как python run.py seed)
show-settings
Показать FLASK_ENV, TEST_MODE, пути контента и др. (без секретов)
list-commands
Показать этот же список в окне сообщения
